# Branding Guidelines

## Current Direction

The current visual direction is restrained and dark by default.

Core palette:

- Background: `#0B0D10`
- Surface: `#12161C`
- Elevated: `#181D25`
- Border: `#2A313B`
- Primary text: `#F2F4F7`
- Muted text: `#98A2B3`
- Accent: `#7C8CFF`

## Rules

- Avoid placeholder screenshots in public-facing materials.
- Prefer SVG sources for logos and installer graphics.
- Keep social graphics minimal and avoid version numbers.
- If branding changes materially, replace outdated screenshots before the next release.
