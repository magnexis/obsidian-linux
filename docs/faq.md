# FAQ

## Is Obsidian Linux ready for daily use?

Not yet. The repository currently targets a reproducible foundation and live ISO workflow rather than a finished end-user distribution.

## Which desktop is implemented right now?

The current implemented target is the Labwc-based core desktop.

## Are Plasma or Hypr editions available?

No. They are future directions only and are not presented as implemented in this repository state.

## Are screenshots in the repository real?

They should be. This project explicitly avoids mockups for operating system screenshots. Any screenshot committed to `docs/images/` must come from a real running build.

## Which hosts are intended for building?

Debian-based amd64 systems, especially Debian 13 or recent Ubuntu LTS releases.
